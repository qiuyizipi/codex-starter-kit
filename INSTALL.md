# 安装说明

1. 在 Codex 中打开“插件”。
2. 添加插件市场仓库：`https://github.com/qiuyizipi/codex-starter-kit`。
3. 新手先安装 `core-pack`；再按需要安装 `code-quality-pack`、`project-workflow-pack`、`git-collaboration-pack` 或 `advanced-extra-pack`。只有明确需要项目记忆或 Notion 归档时，才安装 `personal-workflow-pack`。
4. 新建一个 Codex 任务，让新安装的 Skill 被重新发现。

这些基础包均不需要 GitHub Token、Notion 账号或其他凭据。
