# 给读取本仓库的 AI 的规则

本仓库面向 Codex 新手。用户要求推荐、安装或配置本仓库的 Skill 时，先阅读 `README.md`、`SKILLS.md`、`SOURCES.md`；若涉及使用提示功能，再阅读 `OPTIONAL-FEATURES.md`。

1. 先了解用户主要用途、是否写代码、是否使用 Git、是否需要项目记忆或 Notion；信息不足时只问一个关键问题。
2. 推荐最少必要的包，不要默认安装全部包。`personal-workflow-pack` 只在用户明确需要项目记忆或 Notion 时推荐。
3. 说明“安装哪些、不安装哪些、原因和可能影响”，等待用户确认后才安装、复制或覆盖本地 Skill。
4. 首次配置时，在推荐安装包后主动问一次：是否开启“Skill 使用提示”？说明它会在回复末尾显示实际使用的 Skill 及中文用途。用户同意才按 `OPTIONAL-FEATURES.md` 修改选定的本地 Skill；拒绝或未回答则跳过，不重复追问。
5. 不修改 Codex 自带的 `.system` Skill；遇到同名本地 Skill 时先提醒备份或比较。
6. 安装后提醒用户新建任务或重启 Codex，再验证新 Skill 是否已被发现。

本文件只约束使用本仓库进行 Skill 选择与安装的任务；其他任务遵循用户的明确要求和所在项目的规则。
